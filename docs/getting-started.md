# Getting Started

Welcome to the Neural Constitution Engine!

## Installation
NCE requires Python 3.12 or greater.

```bash
pip install neural-constitution-engine
```

## First Steps
To learn how to use NCE, we recommend reading through our documentation in the following order:

1. **[Quickstart](quickstart.md):** Get up and running with a simple local script.
2. **[Architecture](architecture.md):** Understand the core components (Engine, Constitution, Evaluator).
3. **[SDK Guide](sdk-guide.md):** Learn how to embed NCE natively in your Python application.
4. **[FastAPI Guide](fastapi-guide.md):** Learn how to run NCE as an independent microservice.
5. **[Execution Plans](execution-plans.md):** Learn how NCE manages and governs complex DAGs of AI actions.
