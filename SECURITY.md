# Security Policy

## Supported Versions

The Neural Constitution Engine guarantees support and security patches for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| 0.1.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT report it by opening a public GitHub issue.

Instead, please send an email to **security@example.com** with a detailed description of the vulnerability. We will acknowledge your email within 48 hours and work with you to resolve the issue as quickly as possible.
